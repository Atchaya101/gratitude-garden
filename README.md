# 🌱 Gratitude Garden

A simple and calming gratitude journal where each daily gratitude entry grows into a beautiful flower in your personal garden.

The project encourages mindfulness by letting users plant one gratitude seed every day and watch their garden bloom over time.

---

## ✨ Features

- 🌱 One gratitude entry per day
- 🌸 Every entry grows into a unique flower
- 🔥 Daily streak tracker
- 📖 View previous journal entries
- 💾 Offline data storage using Local Storage
- 📱 Responsive design for desktop and mobile
- 🎨 Clean and relaxing user interface

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript
- Local Storage API
